Acessibility

Essa página contém o atributo lang com o valor pt-br pois o conteúdo encotrado aqui está em português. No entanto, o título "Acessibility" contém um lang lang com um valor en pois se trata de um texto em inglês. Isso auxilia alguns leitores de tela na decisão do sotaque.

Tags semânticas sempre terão um caráter acessível

Imagens sempre devem ter um alt

Isso ajuda leitores de tela entender do que se trata uma imagem. Algumas imagens não precisam de um texto alternativo, mesmo assim, utilize o atributo alt sem um valor.

Elementos interativos devem ter um tabindex

Dessa forma, o usuário pode navegar pela web page pelo teclado. Por padrão, elementos interativos têm o tabindex igual a zero, assim, a navegação será feita do topo da página até o final. É possível trocar a ordem de navegação no entanto (use o tab para trocar o foco dos elementos):
